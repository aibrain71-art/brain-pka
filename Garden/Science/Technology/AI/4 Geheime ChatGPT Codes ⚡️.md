Detailed summary

- Die vier Geheimcodes für [[ChatGPT]] sind SH, DLTR, EL5 und Lystify, die jeweils unterschiedliche Funktionen haben, um die Antworten von ChatGPT zu beeinflussen [(00:00:00)](https://www.youtube.com/watch?v=Dk1VXsxXrt8&t=0s).
- Der Code SH wird verwendet, um die Antworten von ChatGPT menschlicher zu formulieren, indem man "human" vor dem Prompt schreibt [(00:00:09)](https://www.youtube.com/watch?v=Dk1VXsxXrt8&t=9s).
- Der Code DLTR dient dazu, kompakte und gut verständliche Antworten von ChatGPT zu erhalten, indem man DLTR nach dem Prompt schreibt und somit die typischen langen Antworten überspringt [(00:00:17)](https://www.youtube.com/watch?v=Dk1VXsxXrt8&t=17s).
- Der Code EL5 oder EL10 wird verwendet, um die Dinge wie einem Fünf- oder Zehnjährigen zu erklären, indem man EL5 oder EL10 nach dem Prompt schreibt [(00:00:24)](https://www.youtube.com/watch?v=Dk1VXsxXrt8&t=24s).
- Der Code Lystify wird verwendet, um die Antwort als Liste mit gut strukturierten Stichpunkten zu erhalten, indem man Lystify nach dem Prompt schreibt [(00:00:40)](https://www.youtube.com/watch?v=Dk1VXsxXrt8&t=40s).
